Einbetten
Fügen Sie diesen Code in Ihre Website ein
Ihr Einbettungs-Code

Einmalig vor dem schließenden </body> Tag einfügen:

Der Chatbot erscheint als Button unten rechts auf Ihrer Website.

📋 Schritt-für-Schritt Anleitung
🔵
WordPress
+
  1. WordPress Admin öffnen → Design → Theme-Editor
  2. Rechts die Datei footer.php auswählen
  3. Den Code direkt vor </body> einfügen
  4. Datei aktualisieren klicken – fertig!

💡 Alternativ: Plugin "Insert Headers and Footers" installieren → Footer-Bereich → Code einfügen

🟡
Wix
+
  1. Wix Dashboard → Einstellungen → Erweitert
  2. Custom Code auswählen
  3. + Code hinzufügen klicken
  4. Code einfügen → Ort: "Body – Ende" auswählen
  5. Anwenden klicken – fertig!
🔷
Webflow
+
  1. Webflow → Projekteinstellungen (Zahnrad-Icon)
  2. Tab Custom Code öffnen
  3. Den Code im Bereich "Before </body> tag" einfügen
  4. Speichern → Website neu veröffentlichen – fertig!
🟢
Shopify
+
  1. Shopify Admin → Online-Shop → Themes
  2. Beim aktiven Theme auf "..." → Code bearbeiten
  3. Datei theme.liquid öffnen
  4. Den Code direkt vor </body> einfügen
  5. Speichern klicken – fertig!
⬛
Squarespace
+
  1. Squarespace → Design → Code-Injektion
  2. Den Code im Bereich "Fußzeile" einfügen
  3. Speichern klicken – fertig!
🌐
Normales HTML
+
  1. HTML-Datei in einem Texteditor öffnen
  2. Am Ende der Datei </body> suchen
  3. Den Code direkt davor einfügen
  4. Datei speichern und hochladen – fertig!
