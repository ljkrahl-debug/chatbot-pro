Häufige Fragen
Alles was Sie wissen müssen.
Wie lange dauert die Einrichtung? +

Nach Ihrer Bestellung richten wir Ihren Chatbot innerhalb von 48 Stunden ein – vollständig konfiguriert, mit Ihrem Branding und Ihren Texten.

Muss ich technische Kenntnisse haben? +

Nein! Sie fügen einmalig eine Zeile Code in Ihre Website ein – das war's. Das Dashboard bedienen Sie wie eine normale App, ganz ohne Technik.

Kann ich den Chatbot jederzeit kündigen? +

Ja! Es gibt keine Vertragsbindung. Sie können jederzeit mit 14 Tagen Frist zum Monatsende kündigen.

Auf welchen Websites funktioniert der Chatbot? +

Auf allen! WordPress, Shopify, Webflow, Wix, oder einfaches HTML – überall wo Sie eine Zeile Code einfügen können, läuft der Chatbot.

Ist der Chatbot DSGVO-konform? +

Ja! Das Hosting erfolgt in Deutschland, Daten werden nicht an Dritte weitergegeben und Gespräche werden nach 90 Tagen automatisch gelöscht.

Was passiert wenn der Chatbot eine Frage nicht kennt? +

Die KI antwortet trotzdem intelligent und empfiehlt dem Kunden, Sie direkt zu kontaktieren. Sie können jederzeit mehr FAQ-Einträge hinzufügen.

Wann muss ich bezahlen? +

Die Einrichtungsgebühr (199 €) wird erst nach Fertigstellung fällig. Die monatliche Gebühr (29 €) startet mit dem ersten Monat nach der Einrichtung.
